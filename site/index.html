What CTOX is

Persistent autonomy for technical operations.

CTOX runs as a durable agent system around a machine or service: it plans work, continues interrupted missions, routes communication, supervises context, and drives execution until the operational task is actually closed.

Long-running work Mission state, queues, schedules, watchdogs, and follow-ups survive beyond a single agent turn.
Host control Built for build, DevOps, infrastructure, browser, web, and verification workflows on real systems.
Desktop entry point The desktop app gives users a guided local surface and installs the CTOX CLI from inside the product.
