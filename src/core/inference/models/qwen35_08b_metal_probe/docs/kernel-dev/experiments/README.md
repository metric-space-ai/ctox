# Generated Experiment Records

`tools/new_kernel_experiment.sh` writes timestamped experiment records here.
It also regenerates `INDEX.md` after creating a record.

These files are working documents. Promote important results into
`RESEARCH_LOG.md` after the experiment reaches a decision.

Each generated record should include hashes for both the accepted baseline
profile and the metalpack. If either changes, rerun the baseline.

`INDEX.md` is generated by:

```text
tools/update_kernel_experiment_index.sh
```
