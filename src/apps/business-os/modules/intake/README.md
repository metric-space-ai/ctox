# Bewerbungseingang

Generischer Mehrkanal-Eingang: normalisierte Bewerbungen aus Karriereseite/Jobbörse/E-Mail/QR.

Generic Business OS ATS module. Owns the `applications` collection(s); the engine lives in `core/` (pure, unit-tested) and the mount in `index.js` renders a record list + create form wired to that engine. Part of the ATS build-out (`docs/business-os-ats-plan.md`).
