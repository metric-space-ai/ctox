# Einwilligungen

Generisches DSGVO-Einwilligungs-/Rechtsgrundlagen-Register mit Zweck und Löschfrist.

Generic Business OS ATS module. Owns the `business_consents` collection(s); the engine lives in `core/` (pure, unit-tested) and the mount in `index.js` renders a record list + create form wired to that engine. Part of the ATS build-out (`docs/business-os-ats-plan.md`).
